Detailed Description

WARNINGS

These functions are designed to be used in an ISR, Interrupt Service Routines. If you use them in a program with interrupts enabled, access to I/O ports may not be atomic. This will be true for many pins on the Mega so you should place a cli() before calls and a sei() after calls to read functions from interrupt enabled code.

On the Mega you should try to use pins on PORTA - PORTG. These ports will have faster performance and access to I/O ports will be atomic.
